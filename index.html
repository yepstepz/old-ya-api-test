Пример работы библиотеки

	
Тесты

	

	Список основных методов. Подробнее
		
    
			
  • getEntity( [number] ) - получить объект.
    • 
			
  • setEntity({ name: [string] }) - задать объект.
    • 
			
  • editName( [number], [string] ) - отредактировать имя.
    • 
			
  • deleteEntity( [number] ) - удалить объект.
    • 
			
  • addObjectToEntity( [ numbers array ], [ string ], [objects array] ) - добавить объекту данные в параметры
    • 
			
  • changeEntityObject( [ numbers array ], [ string ], [old objects array], [new objects array] ) - изменить у объекта данные в массиве параметров.
    • 
			
  • deleteEntityObject([number] ) - удалить из массива параметров объект.
    • 
			
  • setDate( [number], [ string (dd.mm.yyy) ] ) - задать лекции дату.
    • 
			
  • getDate( [number] ) - получить отформатированную дату лекции.
    • 
			
  • setCount( [number] ) - задать школе количество учеников.
    • 
			
  • getLections( [number] ) - получить лекции, которые относятся к школе.
    • 
			
  • showShedule( [number] ) - показать упорядоченное расписание для школ или аудиторий. Вызов без аргументов покажет все даты.
    • 
			
  • filterSheduleByDate( [array], [string], [string]) - посмотреть раписание для школ или аудиторий в заданный интервал времени. Первый параметр "all" покажет все лекции в интервале.
    • 
			
  • setCapacity( id[number], [number] ) - установить вместимость у аудитории.
    • 
			
  • setAddress( [number], [object {classroom: [number]}]) - задать аудитории адрес.
    • 
			
  • setLections( [number], [array] ) - задать лекции аудитории.
    • 
			
  • deleteLections( [number], [array] ) - удалить у аудитории лекции.
    • 
		

		
Начать использовать:

		
    
			
  • var lection = new Lection() - теперь можно работать с лекциями, назначать лекторов, время и школы.
    • 
			
  • var school = new School() - теперь можно работать со школами, задавать количество учеников и смотреть расписание.
    • 
			
  • var cabinet = new Cabinet() - работа с аудиториями, можно задавать адрес, смотреть расписание и задавать вместительность.
